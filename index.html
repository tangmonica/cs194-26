CS 194-26 Projects

Monica Tang


Colorizing the Prokudin-Gorskii Photo Collection

Prokudin-Gorskii captured thousands of "photos" of the Russian Empire by recording each scene onto glass plates with a red, a green, and a blue filter. This project aims to automatically produce a color image from these RGB plates via image-processing techniques.

Filters and Frequencies

This project uses filters and image frequencies to produce a variety of results, including image sharpening, hybrid images, and multi-resolution blending.

Face Morphing and Modeling a Photo Collection

The first part of this project aims to create an animation that morphs one face into another. The second part uses a set of images of a population to compute the average face for that population.

Auto-Stitching Photo Mosaics

The first part of the project explores perspective warps and mosaic stitching. The second part automates the stitching process.

Facial Keypoint Detection with Neural Nets

The goal of this project is to automatically detect facial keypoints with a deep convolutional neural network. This project uses the PyTorch library.

Image Quilting & Artistic Neural Style Transfer

In the first project, we synthesize larger texture images using small texture samples. We are also able to perform texture transfer, which involves giving an object the appearance of a texture while maintaining its shape. In the second project, we use neural networks to transfer artistic styles to realistic photographs.
